## Database Query Examples

### Create User (Registration)
```sql
-- User registers with email "john@example.com"
INSERT INTO User (id, email, password, name)
VALUES (
  'clyz1abc123...',
  'john@example.com',
  '$2b$10$hashed_password_here...',  -- bcryptjs hash
  'John Doe'
);
```

### Find User (Login)
```sql
-- Find user by email
SELECT * FROM User 
WHERE email = 'john@example.com';

-- bcryptjs then compares entered password with stored hash
```

### Create Conversation
```sql
-- Create conversation between John (id1) and Jane (id2)
INSERT INTO Conversation (id)
VALUES ('conv123abc...');

-- Link participants
INSERT INTO _ConversationToUser (A, B)
VALUES ('conv123abc...', 'user_id_john');

INSERT INTO _ConversationToUser (A, B)
VALUES ('conv123abc...', 'user_id_jane');
```

### Get Messages for Conversation
```sql
-- Fetch all messages for conversation
SELECT m.id, m.content, m.createdAt, 
       u.id, u.name, u.email
FROM Message m
JOIN User u ON m.senderId = u.id
WHERE m.conversationId = 'conv123abc...'
ORDER BY m.createdAt ASC;
```

### Save Message
```sql
-- User sends message
INSERT INTO Message (id, content, senderId, conversationId, createdAt)
VALUES (
  'msg456def...',
  'Hi Jane!',
  'user_id_john',
  'conv123abc...',
  CURRENT_TIMESTAMP
);

-- Update conversation timestamp
UPDATE Conversation
SET updatedAt = CURRENT_TIMESTAMP
WHERE id = 'conv123abc...';
```
